MONOGATARI
==========

##Markdown-based blog.

###Characteristics
* Extended markdown syntax ([php-markdown], [php-smartypants])
* Syntax highliting ([prettify])
* Localization (two languages)
* Caching of rendered markdown pages
* Support for themes
* Simple and short single-file application
